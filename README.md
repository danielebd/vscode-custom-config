# VS Code Custom Config

a. Modificare colori dell'area di lavoro

### Vue
```bash
{
  "workbench.colorCustomizations": {
    "titleBar.activeBackground": "#3FB27F",
    "titleBar.activeForeground": "#fff"
  }
}
```
### Laravel
```bash
{
    "workbench.colorCustomizations": {
        "titleBar.activeBackground": "#ac3535",
        "titleBar.activeForeground": "#fff"
    }
}
```
